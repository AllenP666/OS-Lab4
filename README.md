# Лабораторная работа 4: Мониторинг температуры

Программа для считывания температуры с USB/серийного порта и ведения статистики.

## Сборка
- Windows: `update_build.cmd`
- Linux: `./update_build.sh`

## Запуск (Симуляция)
1. В первом терминале: `./build/sensor_simulator`
2. Во втором терминале: `./build/weather_logger virtual_com`

Программа создаст:
- `raw.log` (хранит данные за 24 часа)
- `hourly.log` (среднее за час, хранит за месяц)
- `daily.log` (среднее за день, хранит за год)
